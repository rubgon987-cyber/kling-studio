# Cómo subir Kling Studio a Hostinger

## Pasos

1. Entra a hpanel.hostinger.com
2. Ve a "Administrador de archivos" o usa FTP
3. Entra a la carpeta: public_html/
4. Sube TODOS los archivos de esta carpeta (kling-studio/)
5. Abre tu dominio en el navegador — ya funciona

## Para FTP (FileZilla)
- Host: tu dominio o IP del hosting
- Usuario y contraseña: los de Hostinger
- Puerto: 21
- Carpeta remota: /public_html/

## Primer uso
1. Abre tu dominio
2. Ve a Configuración (icono ⚙)
3. Pega tu API Key y API Secret de Kling
4. Guarda y empieza a generar videos
